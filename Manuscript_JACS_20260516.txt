ABSTRACT

Catalytic nanoclusters exhibit structural dynamics under reaction conditions, yet simulating their morphology and catalytic performance remains challenging due to the requirement for specialized software expertise and multistep manual workflows. Here we present an agent-mediated framework that enables reaction-environment-dependent nanocluster simulations through smartphone-accessible interfaces using multilingual natural-language and voice interaction. The system integrates mobile text and speech input with domain-specific Multi-scale Operando Simulation Package (MOSP) and Kinetic Monte Carlo (KMC) execution modules, converting chemically meaningful user requests into validated simulation tasks with automatic parameter clarification and direct return of morphology and performance-related outputs. We demonstrate the framework using representative platinum and palladium nanocluster systems under CO oxidation conditions, showing that users can specify temperature, pressure, and gas composition via conversational interfaces and obtain catalyst-state predictions without manual simulation file preparation. Systematic validation against literature data confirms the accuracy of predicted morphologies and kinetic performance metrics, while multilingual tests (Chinese and English, text and voice) demonstrate consistent simulation outcomes across all interaction modes. This framework lowers the operational barrier to physics-based catalyst-state simulations, transitioning from specialist desktop workflows to mobile scientific interfaces for computational catalysis.

INTRODUCTION

Catalytic nanoclusters undergo structural evolution under operando conditions, with their morphology, exposed surface facets, and catalytic properties exhibiting strong dependence on temperature, pressure, and gaseous environment. Predicting reaction-dependent morphological evolution and catalytic performance has emerged as a critical challenge in heterogeneous catalysis, as catalytic behavior is often governed by these dynamic states rather than static idealized structures. Although computational tools for nanoparticle morphology prediction and kinetic modeling have advanced significantly, their practical deployment remains largely confined to specialists due to requirements for manual input file preparation, software-specific parameter knowledge, and coordination of multistep computational procedures. Consequently, catalyst-state simulations remain inaccessible through everyday research interfaces, particularly in mobile contexts where convenient scientific interaction is increasingly desired but rarely supported.

Recent advances in workflow automation and large-language-model-based agents have expanded the range of scientific tasks expressible through conversational interfaces, suggesting that multistep computational procedures can be coordinated through modular tools, planning layers, and execution control without requiring users to manually manage each calculation stage. However, most existing efforts focus primarily on workflow orchestration—including tool chaining, scheduler interaction, and task recovery—rather than enabling chemically meaningful simulations for specific catalytic problems. Interfaces that allow users to directly specify catalyst-state simulations under reaction conditions through natural language or speech remain largely unexplored, particularly when target outputs must preserve domain relevance beyond mere software connectivity demonstration.

This limitation is particularly acute for nanocluster catalysis, where scientifically useful simulations require users to define coupled thermodynamic and environmental variables (temperature, pressure, gas composition, particle identity, simulation mode) while navigating software-specific parameter conventions and multistep execution procedures. Even when underlying physical models are available, the practical barrier to obtaining reaction-environment-dependent morphology and performance outputs remains prohibitively high. A more productive approach would establish a chemistry-grounded interaction layer that translates user intent into validated catalyst-state simulation tasks and returns outputs directly usable for catalytic analysis.

Here we address this gap by developing an agent-mediated framework enabling reaction-environment-dependent nanocluster morphology and performance simulations to be initiated directly from smartphone-accessible mobile interfaces through multilingual natural-language and voice interactions. Rather than exposing users to software syntax, file-level workflow construction, and manual parameter handling, the system translates chemically meaningful user intents into validated MOSP- and KMC-based simulation tasks, automatically clarifies incomplete inputs, executes corresponding workflows, and returns morphology and performance-related outputs in interpretable form. We demonstrate this framework using platinum and palladium nanocluster systems under CO oxidation reaction environments, showing that users can specify temperature, pressure, and gas composition through conversational text or speech on mobile devices without manual simulation file preparation. By lowering the operational barrier to accessing physics-based catalyst-state simulations, this work transitions such calculations from specialist desktop workflows toward broadly accessible mobile scientific interfaces for computational catalysis.

RESULTS AND DISCUSSION

3.1 Agent-Mediated Framework for Catalyst-State Simulation

The proposed framework establishes a smartphone-accessible scientific interface enabling users to initiate catalyst-state simulations through natural language or voice interaction without requiring manual input file preparation or knowledge of software-specific workflows. The system architecture comprises three integrated layers: a mobile interaction layer accepting multilingual text and speech input, an agent-mediated coordination layer translating user intent into validated simulation tasks, and a domain-specific execution layer running MOSP-based multiscale structure reconstruction (MSR) and kinetic Monte Carlo (KMC) simulations.

The mobile interaction layer provides users with a familiar conversational interface where simulation requests can be expressed in natural language, either as text or through voice recognition. Users specify metal identity, reaction environment (temperature, pressure, gas composition), and desired output type without needing to understand underlying computational procedures. The agent-mediated coordination layer, powered by a large-language-model-based orchestration system, recognizes scientific task types (morphology prediction, performance analysis), extracts relevant parameters from user requests, identifies missing information, and initiates clarification dialogues when necessary. This layer also validates that requested simulations are physically meaningful with all required parameters within acceptable ranges.

The execution layer integrates domain-specific simulation modules performing actual calculations. The MSR module generates atomic structures of nanoclusters under specified reaction conditions using thermodynamic stability calculations based on Wulff construction with adsorbate-dependent surface energies. The KMC module simulates kinetic processes outputting turnover frequencies, surface coverage, and other catalytic performance indicators using lattice-based kinetic Monte Carlo methodology. Importantly, users do not interact directly with computational modules; instead, the agent layer automatically constructs necessary input files, invokes appropriate executables, and extracts relevant outputs for presentation back to users.

A key innovation of the chatMOSP system is the comprehensive parameter management and retrieval workflow. The system maintains a curated database (MOSP_database) containing validated parameters for various metal-gas systems, including surface energies, adsorption energies, and interaction parameters obtained from density functional theory calculations and experimental literature. When users request simulations for systems with existing database entries, parameters are automatically retrieved and presented for user confirmation or modification. For systems lacking pre-existing parameters, the system employs a multi-tier literature search protocol: first querying open-access journals (Nature Communications, Science Advances, PNAS, ACS Central Science), then subscription-based top-tier journals (Science, Nature, JACS, Angewandte Chemie), and finally preprint servers. This automated parameter retrieval capability enables scientifically meaningful simulations even when users lack domain-specific parameter knowledge.

The natural language processing layer implements sophisticated task recognition using keyword-based classification with weighted scoring. The system distinguishes between morphology prediction requests ("Show me the Pd cluster structure"), performance analysis queries ("What is the turnover frequency"), and parameter inspection commands ("Display the adsorption energies"). Multilingual support is achieved through parallel keyword sets in Chinese and English, enabling consistent task recognition across languages. Voice input is processed through platform-specific speech-to-text APIs with domain-specific vocabulary optimization for scientific terminology.

The workflow orchestration layer manages the complete simulation pipeline: parameter extraction from user input, database lookup or literature search for missing parameters, user confirmation of extracted parameters, input file generation, execution monitoring, result extraction, and visualization generation. For long-running KMC simulations, the system supports asynchronous task execution with automatic status checking and result delivery upon completion. Users receive notification when simulations complete and can view results directly on their mobile devices through integrated visualization tools that generate both static images and rotating 3D animations.

3.2 Pt Nanoclusters under CO Oxidation: Morphology-Activity Relationships

To demonstrate the framework's capability for investigating structure-activity relationships, we conducted comprehensive simulations for Pt nanoclusters under CO oxidation conditions, motivated by experimental observations of oscillatory behavior in this system. Vendelbo et al. (Nature Materials, DOI: 10.1038/NMAT4033) reported that Pt nanoclusters exhibit morphological oscillations during CO oxidation, with spherical morphologies under high CO partial pressure and faceted structures with prominent surface planes under low CO partial pressure. This experimental system provides an ideal test case for validating the framework's ability to capture reaction-dependent morphological evolution and its consequences for catalytic performance.

Figure 2 illustrates the complete workflow executed entirely through mobile conversational interaction. The user initiated the investigation by requesting: "Show me Pt cluster structures under CO oxidation conditions." The system responded by displaying the complete parameter set from the MOSP_database for Pt-CO oxidation, including surface energies for Pt(111), Pt(100), and Pt(110) facets, adsorption energies for CO and O on each facet, and lateral interaction parameters. The user then modified the temperature parameter to 850 K and requested comparison of cluster morphologies at two different pressures: 150 Pa and 1500 Pa.

[Figure 2: Mobile interface screenshots showing the conversational dialogue (a) and resulting cluster structure visualizations displayed on the smartphone (b). The user requests Pt cluster structures at 850 K under two pressure conditions. Static images and rotating 3D animations are generated and displayed directly on the mobile device.]

The MSR module executed rapidly, generating optimized cluster structures for both pressure conditions within minutes. The resulting morphologies are presented in Figure 3a, revealing a clear pressure-dependent structural evolution. At 1500 Pa, the Pt cluster adopts a more rounded, spherical morphology with reduced facet definition. In contrast, at 150 Pa, the cluster exhibits pronounced faceting with well-defined (111) and (100) surface planes and prominent edges and corners. This morphological trend directly corroborates the experimental observations of Vendelbo et al., validating the framework's ability to capture reaction-dependent structural dynamics.

[Figure 3: (a) Comparison of Pt cluster morphologies at 150 Pa and 1500 Pa at 850 K, showing the transition from faceted to rounded structure with increasing pressure. (b) Turnover frequency (TOF) comparison showing the pressure-dependent catalytic activity. The 1500 Pa condition exhibits more than 10-fold higher TOF compared to 150 Pa.]

To quantify the catalytic consequences of these morphological differences, the user requested kinetic Monte Carlo simulations to determine turnover frequencies for CO oxidation under both conditions. The system displayed the KMC parameter set including reaction events, activation barriers, and pre-exponential factors, which the user confirmed before initiating the simulations. KMC calculations for 20 million steps required approximately 24 hours of computation time, during which the user could engage in other tasks through the mobile interface. Upon completion, the system delivered the results directly to the mobile device.

The TOF analysis (Figure 3b) revealed a striking pressure dependence: the 1500 Pa condition exhibited a turnover frequency more than 10 times higher than the 150 Pa condition. This dramatic activity difference can be rationalized through the combined morphological and kinetic analysis. The rounded morphology at higher pressure provides a higher density of undercoordinated surface sites (steps, kinks, edges) which are known to be more active for CO oxidation. Additionally, the higher CO partial pressure increases the surface coverage of reactive intermediates, further enhancing the reaction rate.

The mechanistic interpretation of the experimentally observed oscillatory behavior emerges naturally from these results. Under high CO partial pressure, the cluster adopts a rounded morphology with abundant undercoordinated sites, leading to rapid CO oxidation and consequent CO consumption. As CO partial pressure decreases due to the fast reaction, the cluster transforms to a faceted morphology with predominantly close-packed surfaces, dramatically reducing catalytic activity. The diminished CO consumption allows CO partial pressure to recover, driving the system back to the rounded high-activity state, thus completing the oscillation cycle. This mechanistic picture, validated through the mobile conversational interface, demonstrates the framework's capability to generate scientific insights beyond simple task execution.

Additionally, we tested the framework using parameters from our previous work (J. Phys. Chem. C 2021, 125, 19756-19762), successfully reproducing Pt cluster structures across multiple temperature and pressure conditions through straightforward conversational parameter modification. These additional results are provided in Supporting Information.

3.3 Pd Nanoclusters under CO Oxidation: Database and Literature-Derived Parameters

We further validated the framework using Pd nanoclusters under CO oxidation conditions, demonstrating two distinct workflows: (1) utilizing pre-existing database parameters for comparison with literature data, and (2) retrieving parameters from web-based literature sources when database entries are incomplete. Both workflows were executed entirely through mobile conversational interfaces.

For the database-driven workflow, we leveraged parameters previously obtained from Chee et al. (Nature Communications, DOI: 10.1038/s41467-020-16027-9), which reported in situ TEM observations of Pd nanocluster structural evolution during CO oxidation. The user requested: "Show me Pd cluster structures under CO oxidation at different temperatures." The system retrieved the complete parameter set from MOSP_database including BEEF-vdW calculated adsorption energies (Eads,CO = -1.666 to -1.734 eV, Eads,O = -0.960 to -1.214 eV across different facets) and interaction parameters, which the user confirmed before execution.

[Figure 4: (a) Mobile interface dialogue showing parameter display and modification for Pd cluster simulation. (b) Temperature-dependent morphological evolution of Pd clusters from 473 K to 873 K. Low-temperature structures exhibit pronounced faceting with well-defined (111) and (100) surfaces, while high-temperature structures become progressively more rounded. The morphological trend shows excellent agreement with experimental TEM observations from Chee et al. (Nature Communications, 2020).]

The resulting temperature-dependent morphological evolution is presented in Figure 4b. At low temperatures (473 K), the Pd cluster exhibits pronounced faceting with well-defined (111) and (100) surfaces and sharp edges. As temperature increases, the structure progressively rounds, with the 873 K cluster approaching a near-spherical morphology with reduced facet definition. This trend shows excellent qualitative agreement with the experimental TEM observations of Chee et al., who reported similar temperature-dependent morphological transitions under equivalent reaction conditions.

To test multilingual and multimodal capabilities, we repeated these simulations using Chinese text input, voice input in both Chinese and English, and different large language models as the orchestration backend. All input modes produced statistically identical simulation results, confirming the framework's robustness across interaction modalities. Representative conversation transcripts for different input modes are provided in Supporting Information.

For the literature retrieval workflow, we deliberately removed critical parameters (adsorption energies and interaction parameters) from the database entry for Pd-CO oxidation, simulating a scenario where users lack pre-existing parameters for their system of interest. The user made the same request: "Show me Pd cluster structures under CO oxidation." The system detected the missing parameters and prompted: "Required parameters are missing from the database. Would you like me to search the literature for these parameters?"

[Figure 5: (a) Mobile interface showing the parameter retrieval workflow. The system identifies missing parameters and presents options for literature search. (b) Literature search results displayed on the mobile device, showing relevant articles with parameter availability indicators. (c) Final Pd cluster structure obtained using literature-derived parameters, showing agreement with structures generated from database parameters.]

Upon user confirmation, the system initiated an automated literature search protocol. The agent queried Nature Communications using the search format "Pd" AND "CO oxidation" AND "adsorption energy", retrieved the top 10 search results, and automatically screened article abstracts and supplementary information for relevant parameter data. The system presented the three most relevant articles to the user with parameter availability indicators, downloaded supplementary information files, and extracted adsorption energies and interaction parameters from tables and text. The extracted parameters were displayed for user confirmation before simulation execution.

The resulting Pd cluster morphology generated using literature-derived parameters showed excellent agreement with the database-driven results, validating the parameter retrieval workflow. This capability significantly extends the framework's utility to systems where users lack domain-specific parameter knowledge, enabling scientifically meaningful simulations through automated literature mining and parameter extraction.

CONCLUSIONS

We have developed an agent-mediated framework that enables reaction-environment-dependent nanocluster morphology and performance simulations through smartphone-accessible mobile interfaces using multilingual natural-language and voice interaction. The system successfully translates chemically meaningful user requests into validated MOSP- and KMC-based simulation workflows, automatically retrieves or searches for required parameters, executes calculations, and returns interpretable results without requiring users to prepare simulation files or possess software-specific expertise.

Validation studies using Pt and Pd nanoclusters under CO oxidation conditions demonstrate that the framework produces morphologies and kinetic performance metrics in excellent agreement with literature data. The Pt system investigation revealed pressure-dependent morphological evolution consistent with experimentally observed oscillatory behavior, while the Pd system validation confirmed the accuracy of both database-driven and literature-retrieval parameter workflows. Multilingual and multimodal tests confirmed consistent performance across Chinese and English text and voice inputs.

This framework does not replace underlying physical models but rather lowers the operational barrier to accessing them, transitioning catalyst-state simulation from specialist desktop workflows toward mobile scientific interfaces for computational catalysis. The combination of natural language accessibility, automated parameter management, and rigorous validation against established literature establishes a new paradigm for democratizing computational catalysis tools while maintaining scientific rigor.
